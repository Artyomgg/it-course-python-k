Видео урок

Для разработки программ используются среды программирования (интегрированные среды разработки, ИСР, англ. IDE, Integrated Development Environment).

Среда программирования — комплекс программ, используемых при разработке программного обеспечения.

Для работы с языком програм мирования Python можно исполь-Python можно исполь использовать среды программирования:

  • среда IDLE устанавливается вместе с языком Python
  • среда PyCharm
  • онлайн-среды программирования

Для работы с python можно использовать профессиональные среды разработки и редакторы кода, но мы будем использовать онлайн редактор.
