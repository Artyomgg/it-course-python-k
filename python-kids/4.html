Компьютерный исполнитель

Видео урок

Некоторые команды исполнителя Черепаха, с помощью которых можно написать программы, представлены в таблице

Для того чтобы Черепаха могла выполнять команды, первой командой в программе нужно подключить модуль, содержащий команды управления Черепахой: import turtle

Каждая команда управления Черепахой записывается после ключевого слова turtle и отделяется от него точкой. Команды, записываемые после точки, можно выбирать из выпадающего списка. Для отображения списка используется комбинация клавиш CTRL + пробел (пример 18.12). Команды Черепахи можно копировать, вырезать и вставлять, так же как это делали в текстовом редакторе. При использовании сочетаний клавиш: Ctrl + C, Ctrl + V, Ctrl + X, Ctrl + Z — должен быть включен английский язык.
